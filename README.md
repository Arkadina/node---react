# NodeJS & React

Integrating ReactJS with NodeJS in localhost. Requests are provided by axios and the data is storaged through mysql.

<img src="https://i.ibb.co/61DvPvB/img2.png"/>


## Libraries

styled-components, mysql, express, axios, cors.

## Start server

Client

```bash
npm install
node index.js
```

Server
```bash
npm install
npm start
```
